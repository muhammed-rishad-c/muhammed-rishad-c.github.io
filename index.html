ContextEngine
Contributor Repo ↗

An open-source memory framework for LLM applications. Landed three merged pull requests adding async support, accurate token accounting, and shared utilities across memory backends.

  • Centralized duplicated cosine-similarity logic into a reusable utility module, cutting duplication across multiple memory backend implementations.
  • Built a pluggable TiktokenEstimator using OpenAI's tiktoken library for accurate context-budget enforcement across GPT-3.5, GPT-4, and Claude model families.
  • Designed AsyncContextEngine using asyncio.to_thread(), adding seamless async/await support for FastAPI and asyncio-based apps without touching the existing sync API.
  • Expanded automated test coverage for async execution, concurrency, token estimation, and edge-case handling.
DSA Profiler
Author & Maintainer Repo ↗ PyPI ↗

A published Python package for benchmarking algorithm execution time, memory consumption, and empirical time-complexity analysis.

  • High-precision runtime and memory profiling using time.perf_counter() and tracemalloc.
  • Statistical complexity estimation using Ordinary Least Squares (OLS) regression and Pearson's R² correlation.
  • Lightweight SQLite persistence layer for historical benchmark storage and longitudinal analysis.
  • Packaged to modern Python standards with automated tests and full documentation, published on PyPI.
